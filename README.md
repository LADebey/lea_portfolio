# Portfolio Léa Grivel

Portfolio web premium pour **Léa Grivel** — *UX Researcher & Designer orientée stratégie*.

Inspiré de l'architecture Bento Grid du template **Villo**, ce projet met en valeur l'investigation utilisateur, la stratégie DEX, la résilience et le design d'expérience éthique.

## 🛠️ Stack Technique
- **Framework** : [Next.js](https://nextjs.org) (App Router, React 19)
- **Styling** : [Tailwind CSS v4](https://tailwindcss.com)
- **Animations** : [Framer Motion](https://framer.com/motion)
- **Icônes** : [Lucide React](https://lucide.dev)

## 🎨 Fonctionnalités
- **Thèmes Dynamiques** : Bascule entre les deux palettes personnalisées :
  - **Thème A (Noir & Rose)** : Fond `#262626`, accent `#DB98A1`
  - **Thème B (Gris & Violet)** : Fond `#A9A9A9`, accent `#5A3096`
- **Architecture Bento Grid** :
  - Carte d'introduction & commutateur de thème
  - Portrait photo de Léa Grivel avec halo réactif
  - Horloge temps réel Paris (GMT+2)
  - Visualiseur musical animé (*Lost - Linkin Park*)
  - Parcours & Expériences (KPMG France, DN Inforeso) & Formations
  - Compétences interactives (Hard Skills & Soft Skills)
  - Projets phares & Bouton de téléchargement du CV (PDF)

## 🚀 Démarrage Rapide

Pour lancer le serveur de développement local :

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

### Build de Production

```bash
npm run build
```
